# スキル説明文の表記ルール

SkillMaster.json の description フィールドを書く際のルール。

## 基本方針

- 平易な日本語で、ゲームに馴染みのない人でも理解できる表現を使う
- 専門用語や略語は避け、具体的に説明する
- 数値はそのまま記載し、計算式は書かない

## 文法ルール

### 修飾語の位置

修飾語は修飾する対象の直前に置く。

- ✗ 必ず攻撃してきた敵に反撃する
- ○ 攻撃してきた敵に必ず反撃する

### 不要な接続詞

「ただし」「しかし」などの接続詞は省略できる場合は省略する。

- ✗ 敵に反撃する。ただし、攻撃回数は通常の30%。
- ○ 敵に反撃する。攻撃回数は通常の30%。

### 動詞の統一

「行う」ではなく「する」を使う。

- ✗ 追加攻撃を行う
- ○ 追加攻撃をする

### 自動処理の表現

スキル効果は自動で発動する。選択の余地を示す表現は使わない。

**「まで」は使わない**

- ✗ 魔法ダメージを3回まで無効化する
- ○ 魔法ダメージを3回無効化する

**「できる」ではなく「する」**

- ✗ 20%の確率で2回行動できる
- ○ 20%の確率で2回行動する

### 能動態で統一

効果は「〜が〜する」の形で書く。自分を含む対象でも能動態。

- ✗ 味方全員が受けるダメージを5%減少させる
- ○ 味方全員が受けるダメージが5%減少する

「させる」は使わない。スキルが自動的に効果を発揮するので「する」で統一。

### 読点

可読性を上げない読点は入れない。短い文に読点は不要。

- ✗ 20%の確率で、1ターンに2回行動する。
- ○ 20%の確率で1ターンに2回行動する。

読点を入れる場合：
- 長い修飾句の区切り
- 同じ助詞が連続する場合（「に」「で」など）

同じ助詞の連続を避ける：
- ✗ 敵を倒した時に敏捷に応じた確率で追加攻撃をする
- ○ 敵を倒した時、敏捷に応じた確率で追加攻撃をする

### 助数詞

数量には適切な助数詞をつける。

- ✗ 行動回数が1増加する
- ○ 行動回数が1回増加する

### 効果の持続期間

累積効果や一時効果の持続期間を明示する。

- ✗ 効果は累積する（いつまで？）
- ○ 効果は戦闘終了まで累積する

- ✗ 効果は1ターンで消える
- ○ そのターンのみ有効

### 確率表現の冗長さ

「1回だけ〜する」と書けば確定であることは伝わる。「確定で」は不要。

- ✗ 1回だけ確定で蘇生する
- ○ 1回だけ蘇生する
- ○ 100%の確率で蘇生する（確率を明示したい場合）

### ダメージ種別の明示

どのダメージタイプに影響するか明示する。種別不明の表現は避ける。

- ✗ 攻撃で与えるダメージが1.5倍になる
- ○ 物理攻撃で与えるダメージが1.5倍になる
- ○ すべての攻撃で与えるダメージが1.5倍になる

種別を問わずダメージを受けることを示す場合は「ダメージを受けた時」を使う。

### 曖昧な用語の具体化

ゲーム内で意味が定まっている用語は具体的に書く。

- ✗ ペナルティを解除する
- ○ 状態異常を解除する / デバフを解除する

- ✗ スキルを記憶した状態で
- ○ スキルを思い出した状態で

### 発動タイミングの明確化

「〜の味方を」ではなく「〜になった時」でタイミングを明確にする。

- ✗ 戦闘不能の味方を救出する（それ以前に死亡した味方も含むように読める）
- ○ 戦闘不能になった時に救出する

### 「〜で〜で」の回避

同じ助詞「で」が連続する場合は言い換える。

- ✗ 10%の確率で魔法で反撃する
- ○ 10%の確率で魔法反撃をする

## ダメージ表現

| 避ける表現 | 使う表現 |
|-----------|---------|
| 与ダメージ | 攻撃で与えるダメージ |
| 被ダメージ | 受けるダメージ |
| 与ダメ / 被ダメ | 上記の正式表現を使う |

### 例

- ✗ 物理与ダメージ+10%
- ○ 物理攻撃で与えるダメージが10%増加する

- ✗ 被ダメ-15%
- ○ 受けるダメージが15%減少する

## 増減の表現

| 効果 | 表現 |
|------|------|
| 攻撃力が上がる | 「増加する」「上昇する」 |
| 攻撃力が下がる | 「減少する」「低下する」 |
| 受けるダメージが減る | 「軽減する」 |
| 受けるダメージが増える | 「増加する」 |

### 倍率の表現

| データ値 | 表現 |
|---------|------|
| multiplier: 1.1 | 1.1倍になる |
| multiplier: 0.3 | 通常の30%になる |
| percent: +10 | 10%増加する |
| percent: -10 | 10%減少する / 10%軽減する |

## 確率の表現

| 確率 | 表現 |
|------|------|
| 100% | 「必ず」または「確定で」 |
| 50%以上 | 「X%の確率で」 |
| 50%未満 | 「X%の確率で」 |
| 0% | 「発動しない」 |

### 例

- 100%で反撃 → 必ず反撃する
- 30%で追加攻撃 → 30%の確率で追加攻撃する

## 反撃・追撃の説明

反撃や追撃スキルは、以下の要素を含める：

1. **発動条件**（いつ発動するか）
2. **発動確率**（何%で発動するか）
3. **対象**（誰に攻撃するか）
4. **ダメージタイプ**（物理/魔法/ブレス）
5. **性能差**（通常攻撃との違い）

### 性能差の表現

反撃・追撃は通常攻撃より弱いことが多い。この差は「通常の〇%」で表す。

| パラメータ | 説明 |
|-----------|------|
| attackCountMultiplier: 0.3 | 攻撃回数は通常の30% |
| criticalRateMultiplier: 0.5 | 必殺率は通常の50% |
| accuracyMultiplier: 1.0 | （通常通りなら省略可） |

### 例

```
物理攻撃を受けた時に攻撃してきた敵へ必ず反撃する。攻撃回数は通常の30%、必殺率は通常の50%。
```

## ステータス依存の表現

ステータスに依存する効果（statScale）は「〜1につき〜%」形式で表現する。

| statScale | 表現 |
|-----------|------|
| stat: strength, percent: 0.5 | 力1につき0.5% |
| stat: wisdom, percent: 0.8 | 知恵1につき0.8% |
| stat: spirit, percent: 0.3 | 精神1につき0.3% |

### 例

- ✗ 力に応じた確率で発動する
- ○ 力1につき0.5%の確率で発動する

- ✗ 精神に応じてダメージが増加する
- ○ 精神1につき0.5%、ダメージが増加する

## タイミング表現

| トリガー | 表現 |
|---------|------|
| selfDamagedPhysical | 物理攻撃を受けた時 |
| selfDamagedMagical | 魔法攻撃を受けた時 |
| selfEvadePhysical | 物理攻撃を回避した時 |
| allyDefeated | 味方が倒された時 |
| allyDamagedPhysical | 味方が物理攻撃を受けた時 |
| enemyDefeated | 敵を倒した時 |
| turnElapsed | 毎ターン |
| battleStart | 戦闘開始時 |
| turnStart | ターン開始時 |
| turnEnd | ターン終了時 |

## 対象表現

| ターゲット | 表現 |
|-----------|------|
| self | 自分 |
| attacker | 攻撃してきた敵 |
| killer | 味方を倒した敵 |
| randomEnemy | ランダムな敵1体 |
| allAllies | 味方全員 |
| allEnemies | 敵全員 |

## ダメージタイプ

| タイプ | 表現 |
|--------|------|
| physical | 物理攻撃 / 物理ダメージ |
| magical | 魔法攻撃 / 魔法ダメージ |
| breath | ブレス攻撃 / ブレスダメージ |
| all | すべての攻撃 |

## 戦闘用語

| 英語 | 日本語 |
|------|--------|
| barrier | 結界（ダメージを無効化） |
| parry | パリィ |
| shieldBlock | 盾防御 |
| critical | 必殺 |
| proc | 特殊効果の発動 |
| berserk | 暴走 |
| dodge / evasion | 回避 |
| accuracy | 命中 |

## ステータス名

正式名称を使用する。略称は使わない。

| 略称 | 正式名称 |
|------|---------|
| HP | HP（そのまま） |
| 攻 | 攻撃力 |
| 防 | 防御力 |
| 魔 | 魔力 |
| 精 | 精神 |
| 速 | 速さ |
| 命 | 命中 |
| 回 | 回避 |
| 運 | 運 |

## 説明文の構成

1. **効果の概要**（1文で何が起きるか）
2. **補足情報**（性能差など、必要なら）

### 良い例

```
物理攻撃で与えるダメージが10%増加する。
```

```
物理攻撃を受けた時に攻撃してきた敵へ必ず反撃する。攻撃回数は通常の30%、必殺率は通常の50%。
```

```
戦闘開始時に物理ダメージを1回無効化する結界を張る。
```

```
20%の確率で1ターンに2回行動する。
```

```
味方全員が受けるダメージが5%減少する。
```

### 悪い例

```
物理与ダメ+10%  ← 略語、説明になっていない
```

```
被弾時100%で反撃、attackCount×0.3  ← 略語、英語混在
```

```
必ず攻撃してきた敵に反撃する  ← 「必ず」の位置が不適切
```

```
20%の確率で、1ターンに2回行動できる。  ← 不要な読点、「できる」は不適切
```

```
魔法ダメージを3回まで無効化する。  ← 「まで」は不要
```

```
味方全員が受けるダメージを5%減少させる。  ← 「させる」ではなく「する」
```
