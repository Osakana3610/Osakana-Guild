# スキル効果の共通ルール

スキル名・説明・実装の整合に使う共通ルール。
スキルの正本は `SkillVocabulary.md` と `SkillDescriptionStyle.md` を優先する。

## 用語の前提

- **MPは使用しない**。魔法の使用回数は「魔法の使用回数」と表記し、「呪文チャージ」は使わない。
- 説明文・ログ・L10nは `SkillVocabulary.md` の正本語彙に揃える。
- ゲーム内の表記は「リアクション処理」を使わず、**反撃/追撃/報復/再攻撃**を使う。

## 加算/乗算の重複ルール

- **+X% は合算してから乗じる**
  - 例: +10% と +20% → 合計+30% → 係数 `1.3`
- **×Y はそれぞれ乗算する**
  - 例: ×1.1 と ×1.2 → `1.1 × 1.2`
- **+X% と ×Y が併存する場合**
  - `(1 + 合計%/100) × (各倍率の積)` を適用する

## ステータス表記

- 「防御」は**物理防御**を指す。魔法防御は「魔法防御」と明記する。
