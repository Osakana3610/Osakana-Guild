ビルドの組み合わせ

特徴

  • 本格ハクスラ - 装備・タイトル・宝石を組み合わせて最強のビルドを目指そう
  • 放置で進行 - 一斉出撃で複数パーティが自動探索。忙しくても冒険は止まらない
  • ドロップの快感 - 敵ごとに設定されたレアドロップ。タイトル付きアイテムを狙え
  • 個性システム - 冒険者ごとに異なる個性。長所を活かし、短所を補う編成が鍵

ゲームシステム

  • ギルド運営 - 18種族から冒険者を採用し、パーティを編成。死亡した仲間は蘇生で復活できる。
  • ダンジョン探索 - 72のダンジョンを攻略。フロアごとにイベントや敵が待ち受ける。
  • 装備強化 - アイテム合成・宝石改造・タイトル継承で装備をカスタマイズ。
  • ストーリー - 章ごとに語られる物語。進行でダンジョンがアンロックされる。
