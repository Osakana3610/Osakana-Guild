# おさかなギルド

おさかなギルド は Swift で開発中のダンジョン攻略型・放置系RPGプロジェクトです。バトル、スキル、ドロップ計算などのゲームロジックを中心に構築しています。

## 概要

- キャラクター育成と装備ビルド
- 自動進行のダンジョン探索と戦利品獲得
- スキル効果や各種ボーナスの計算ロジック
- ユニットテストによるコアロジックの検証

## 要件

- Xcode 15 以降
- iOS 17.1 以降（動作対象）
- macOS 14 以降（開発環境）

## ビルド方法

```bash
# プロジェクトを開く
open Epika.xcodeproj

# コマンドラインビルド（任意）
xcodebuild -scheme Epika build
```

## ディレクトリ構成（抜粋）

- `Epika/` : アプリおよびゲームロジック
- `EpikaTests/`, `EpikaUITests/` : テストコード
